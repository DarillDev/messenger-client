# Тестовое задание — Frontend Middle разработчик

Реализовать приложение для обмена текстовыми сообщениями.

## Технологические требования

- Angular >=19
- State Manager
- RxJS

Дизайн и остальные технологии — на усмотрение кандидата.

## Функциональные требования

Все страницы не должны быть доступны без авторизации.

Приложение должно состоять из компонентов.

Верстка должна быть адаптивной для экранов 1024px - 1980px

Наличие модульных тестов.

### Страницы

| Роут     | Описание             |
| -------- | -------------------- |
| `/`      | Главная страница     |
| `/login` | Страница авторизации |
| `/user`  | Профиль пользователя |

### Требования

- Все страницы закрыты авторизацией (guard)
- Приложение состоит из компонентов
- Адаптивная верстка: 1024px — 1920px
- Фейковый REST API для работы с сервером чатов
- Окно авторизации показывает ошибки при неверном логине/пароле
- Модульные тесты
